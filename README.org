[[https://huggingface.co/datasets/lnajt/fictional-characters][Data (HuggingFace)]]

* Posts

** [[file:blogpost/post1-register-detector.org][Post 1: Red-Teaming the Assistant Axis with Evil Assistants]]

The assistant axis has a strong register/style component. Adversarial characters designed to sound like helpful assistants (e.g., "Triage Ethicist," "Helpful Disposal Guide") score high on the axis despite producing harmful content. Clamping against the axis removes stylistic veneer but doesn't reliably prevent harmful compliance from these characters.

** [[file:blogpost/post2-persona-space-geometry.org][Post 2: Low-Dimensional Persona Space]]

Persona space is low-dimensional (effective rank ~8 for 1,268 characters), and this holds across universes, question batteries (Lu's 240 vs BiGGen-Bench 90), and within individual franchises. Role space covers only ~43% of character variance. The low dimensionality is a property of the model, not an artifact of question choice.

* Dataset

1,268 fictional characters from 12 universes (Shakespeare, Greek/Chinese/Hindu/Norse/Egyptian mythology, Star Wars, Marvel, Harry Potter, Game of Thrones, Lord of the Rings, Naruto) plus ~287 canonical characters. Role prompts are minimal: just character name and source.

* Reproducing from Scratch

** Requirements

- GPU cluster with 4x 80GB GPUs (e.g., 4x A100/H100) for Qwen3-32B with tensor parallelism
- Python 3.10+
- SLURM (for cluster) or local GPU setup

** Extraction

#+begin_src bash
# SLURM
sbatch scripts/slurm_extract.sh

# Or locally
./scripts/run_extraction.sh --model Qwen/Qwen3-32B --gpus 4
#+end_src

Generates 240 responses per character, extracts activations, saves to ~outputs/~.

** Analysis

#+begin_src bash
python src/analysis/fictional_character_pca.py \
    --activation-dir outputs/qwen3-32b_<timestamp>/activations \
    --output-dir results \
    --layer 32
#+end_src
